Web Blitz

WebBlitz is an exhilarating web design competition featuring two challenging rounds. The first round, Code in the Dark, is a 20-minute challenge where participants code without seeing the monitor, relying solely on their instincts and skills. This round emphasizes logic and accuracy, with participants using a provided coding environment to complete a task. In round 2, Design a Webpage (Themebased), challenges participants to create a webpage based on a theme provided, focusing on creativity and adherence to the theme within a 40-minute duration.

Rules:

  • Participants must adhere to the given time limits.
  • Plagiarism or copying code will lead to disqualification.
  • Any form of cheating will result in immediate disqualification.
  • Judges' decisions will be final and binding.
  • Event organizers reserve the right to disqualify any participant for violating the rules.
